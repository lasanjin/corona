# Corona statistics
Corona statistics for Sweden. Runs currently only with Python 3+.

## Demo
<img src="demo.gif" width="640">

## How to run
Data from [Johns Hopkins CSSE](https://github.com/CSSEGISandData/COVID-19)

 - List countries
```
$ ./corona.py -l
```

 - List sum (globally confirmed)
```
$ ./corona.py -s
```

 - List country
```
$ ./corona.py <country>
```
